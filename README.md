
# Lumina GIF Studio

Lumina GIF Studio 是一个轻量级的、基于浏览器的创意工具，专注于高质量视频处理与图像艺术化渲染。

## ✨ 核心功能

- **GIF 制作**: 将视频片段转换为高质量 GIF，支持精确的裁剪范围、自定义 FPS 和分辨率设置。
- **One Last Kiss 渲染**: 应用电影级的艺术滤镜，具备充满活力的色差效果和铅笔素描纹理。支持专门的“浮雕”模式，用于创作高对比度的黑色风格艺术。
- **幻影坦克合成**: 合成独特的“潜行图片”，使图像在白色和黑色背景下呈现完全不同的内容。
- **响应式设计**: 完美适配桌面与移动端，随时随地进行创作。

## 🛠 技术栈

- **框架**: React 19
- **构建工具**: Vite 6
- **样式**: Tailwind CSS v4
- **动画**: Framer Motion
- **GIF 引擎**: [gifenc](https://github.com/mattdesl/gifenc)
- **图标**: Lucide React

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产环境构建
npm run build
```
